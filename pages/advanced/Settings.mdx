## Settings
Chattabot Settings

## Prefix 
The prefix is the prefix before a command name to run a command.
Change the prefix to a custom one.

### Custom Commands
Add a custom command that chattabot will add to your server.

You can change a command to an embedded by pressing the slider
Each command has a name, description, response
#### Title 
The title of the embed message

#### Description
The description of the embedded message

#### Response
The response that chattabot will send after command execution.

### Disabled Commands
Commands you wish to disable
