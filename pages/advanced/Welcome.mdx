## Welcome
Send a welcome to users that joined your server.

### Channel
The channel to send welcome embed.

### Title
The title of the embed.

### Description

The description on the embed.

### Footer

Change the footer on the description.

### Hex
The colour that the embed will be changed too.
Selection below

### Image URL
Add an Image to the embed, find an image you want to use and add the URL to the field.

### User Profile
Put the user profile next to the footer.

### Send Leaves
Send user leaves to a selected channel.
