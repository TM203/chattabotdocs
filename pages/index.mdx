Chattabot can manage your server, add roles on join, meme votes, button roles and more!

## Permissions Set Up Channels

If a channel is private or restricted by roles, Chattabot can't manage it. Use the [administrator link](https://discord.com/oauth2/authorize?client_id=772746588434595860&permissions=8&scope=bot+applications.commands) or grant Chattabot necessary permissions.
